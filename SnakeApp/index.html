Description and Rules

This is a browser-based SnakeGame in JavaScript. This game does not store any data to local storage. That is, all Scores will clear upon page refresh. May not be playable on mobile devices.

The following are the rules of the game

  • Use LEFT, UP, RIGHT, DOWN arrow keys to move snake.
  • Do not hit the walls of the field
  • Do not eat yourself.
  • Speed of the game increases with scores
